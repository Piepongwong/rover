Example Code - Rover Assignment

I've written 3 different versions for this assignment. Ranging from basic to more advanced:

  1. app.js
  2. appNoConstructor.js
  3. appNoMethods.js

App.js is the most advanced. It includes the bonus exercises and uses a constructor and methods. appNoConstructor.js is using methods, but no contructor. appNoMethods.js is neither using a constructor nor methods. app.js is the best solution because it prevents code repetition and groups functions nicely with the data they influence with methods.
